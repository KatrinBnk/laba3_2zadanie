    Четырехугольники: квадрат(3/13) — прямоугольник(4/14) — ромб(5/15)
2. Работа с "Контейнером" -> buttonKey == 8
3. Добавлена возможность перемещение объектов контейнера при помощи "стрелочек"
4. Список всех функций в main используемых в этой ЛР: //стар == дополнение к предыдущим ЛР
    Работа с панелью кнопок:
    -> back() — логика работы кнопки назад (стар)
    -> createPanel() — обновление или первичное создание панели с кнопками (стар)
    Работа с фигурами на холсте
    -> createContainerResult() — заполнение контейнера фигурами (стар)
    -> MoveContainer(dx, dy) — перемещение фигуры на холсте(стар)
    -> deleteAll() — уничтожение контейнера (стар)
    -> eraseAll() — стирание объектов с холста (стар)
    -> showAll() — рисование объектов на холсте (стар)
    -> changeSize() — изменение размеров ЧЕТЫРЕХУГОЛЬНИКОВ (стар)
    -> changeRadius() — изменение радиусов ОКРУЖНОСТЕЙ (стар)
    -> hierarchy() — работа с подтипами объектов из контейнера
    -> panelORnext(int key) — определение привязок кнопок Окружности и Четырехугольник
